# Chronicle

Chronicle is an early protocol proposal for costly trust signaling on the internet.

Participants publish signed orientations backed by publicly verifiable sacrifice. Nodes relay, batch, anchor, and publish these events so downstream services can compute public trust and reputation from a shared graph.

This repository hosts:
- the whitepaper source
- the generated PDF
- the website source for chronicle-network.org
