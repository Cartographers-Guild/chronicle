# Chronicle

Chronicle is an early protocol proposal for costly trust signaling on the internet.
This repository hosts the whitepaper and website source for [chronicle-network.org](https://chronicle-network.org).

- Whitepaper: [Bringing Trust and Accountability to the Web](./public/chronicle.pdf)
- Essay: [Compass for the Good](https://cartographersguild.substack.com/p/a-compass-for-the-good)
