# `web:url`

## Summary

`web:url` identifies a specific web resource by its canonical absolute URL.

## Subject

`subject` MUST be of type [`absolute_url`](../../../../types/absolute_url/v1.0.0/spec.md).

## Motivation

Links are one of the main ways people encounter unknown territory online. `web:url` makes it possible to judge one specific destination on its own merits.

Clients may use it to distinguish links that are safe, deceptive, useful, misleading, or especially worth opening, sharing, or ranking higher in a feed, forum, search result, or browser UI.
