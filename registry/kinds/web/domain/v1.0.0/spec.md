# `web:domain`

## Summary

`web:domain` identifies a web domain by its canonical domain name.

## Subject

`subject` MUST be of type [`domain_name`](../../../../types/domain_name/v1.0.0/spec.md).

## Motivation

A domain is one of the main public identity surfaces on the web. `web:domain` makes it possible to judge a site, service, publisher, merchant, or operator more broadly than any one specific page or link.

Clients may use it to distinguish domains that appear trustworthy, scammy, abusive, or consistently valuable.
